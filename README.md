# IPAM

## Endpoints
* Allocate an IP range to a VNET
* Allocate a range out of VNet address space
* Free ip range
* Delete allocation
* Scan for ip changes in cloud (Azure only)
* List used/free ip in allocated range

## Features
Support IPV4/IPV6
SSO
Bulk import: CSV,JSON,

## storage
* Azure table storage
* JSON xml file
* TODO: SQL
